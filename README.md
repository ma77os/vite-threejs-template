-   basic scene with plane, sphere, direction light, soft shadows
-   orbit controls
-   gallery: plane with MeshPhysicalMaterial

    -   normals generated via MeshDepthMaterial
    -   canvas as texture of the plane
    -   bulk load images
    -   compose images into canvas
